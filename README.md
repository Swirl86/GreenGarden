# GreenGarden

Prototype developed for self watered plant with output on web-page:

- Python Flask
- HTML-page
- Raspberry pi
- Moisture Sensors
- Arduino
- Water pumps

### Installation

```sh
> npm install flask
```

### Run command

```sh
> python main.py
```

#### Runs on

```sh
localhost:8080
```
